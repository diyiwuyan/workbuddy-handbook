# 附录 E 常见问题与避坑指南

### 安装与启动

| 问题 | 解决方法 |
|---|---|
| 安装后无法启动 | 检查系统版本是否满足要求；Windows 需右键以管理员身份运行安装包 |
| macOS 提示"无法检查恶意软件" | 系统设置 → 隐私与安全 → 安全性栏点击「仍要打开」|
| 登录后没反应 | 检查网络连接；尝试切换登录方式（微信 / 企业微信 / QQ）|

---

### 权限与安全

| 问题 | 解决方法 |
|---|---|
| 不知道 AI 能碰哪些文件 | 初始化时仅勾选工作相关目录；在设置中查看已授权目录列表 |
| Craft 模式下误删文件 | 开启高危操作二次确认；重要文件提前手动备份 |
| 远程操控安全性 | 仅允许个人微信发送指令；定期检查执行记录 |

---

### 自动化

| 问题 | 原因 | 解决方法 |
|---|---|---|
| 定时任务没有触发 | 客户端退了 / 电脑休眠了 | 保持 WorkBuddy 运行，配置锁屏远程 |
| Prompt 写得太模糊，AI 跑偏 | 没指定输入来源和输出格式 | Prompt 里必须包含四要素 |
| 一次性任务永不触发 | 漏填 `scheduledAt` 字段 | once 类型 → 必填 `scheduledAt` |
| 高频任务消耗大量积分 | `FREQ=HOURLY` 一天跑 24 次 | 日常任务用 `FREQ=DAILY`，最多一天一次 |
| 文件找不到 | CWD 设成了 `/tmp` | CWD 设专门的持久工作目录 |

---

### 记忆与上下文

| 问题 | 解决方法 |
|---|---|
| AI 不记得上次聊了什么 | 未开启"生成对话记忆"；或记忆文件被清空 | 在设置中开启记忆功能 |
| 记忆太多，AI 回答变慢 | 记忆条目超过 20 条，消耗大量 Token | 每周清理一次 `MEMORY.md`，删掉过时信息 |
| A 项目的偏好被带到 B 项目 | 所有项目共享同一套记忆，无项目级隔离 | 手动在 Prompt 中指定项目上下文 |

---

### 技能与专家

| 问题 | 解决方法 |
|---|---|
| Skill 触发不了 | 触发词不匹配；或 Skill 未启用 | 在 Skills 管理页面确认 Skill 已启用 |
| 专家输出不专业 | 只是说"你是 XX 专家"，没有真正加载专家模块 | 通过专家中心正式"召唤"专家 |
| 安装的 Skill 太多，AI 变笨 | 每个 Skill 占上下文，过多会分散注意力 | 仅保留最常用的 10-20 个 Skill |

---

### 模型与积分

| 问题 | 解决方法 |
|---|---|
| 积分消耗太快 | 重度任务频率太高；或用了高消耗模型 | 轻量任务用混元；重度任务降低到 WEEKLY |
| 模型响应慢 | 选择了推理型模型处理简单任务 | 日常任务换回混元或 GLM |
| 想用自定义模型 | WorkBuddy 支持 OpenAI 兼容协议 API | 在设置 → 模型选项中配置自定义 API |
